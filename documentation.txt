# Project Documentation

## Project Overview
This project is a professional web application with user authentication functionality. It features a modern UI design and secure backend implementation for user registration and login.

## Technologies Used
- PHP 7.4+
- MySQL Database
- Bootstrap 5.3.6
- Font Awesome 6.0.0
- HTML5/CSS3
- JavaScript

## Project Structure
The project follows a clean file organization:

1. **index.php** - The main landing page with login functionality
2. **signup.php** - User registration page 
3. **home.php** - Dashboard that users see after successful login
4. **connect.php** - Database connection configuration
5. **logout.php** - Handles user logout functionality
6. **check_table.php** - Utility file to check database table structure

## Database Structure
The application uses a MySQL database named 'signupforms' with a table called 'registration' that has the following structure:
- id (INT, AUTO_INCREMENT, PRIMARY KEY)
- username (VARCHAR)
- password (VARCHAR)

## Security Implementations
- Prepared statements used to prevent SQL injection attacks
- Session management for authentication
- Protected routes that require login
- Password handling (Note: In a production environment, passwords should be hashed using proper methods like password_hash())

## Installation Instructions

1. Install XAMPP (or similar PHP development environment)
2. Clone/download the project to the htdocs folder
3. Start Apache and MySQL services
4. Create a database named 'signupforms'
5. Create a table named 'registration' with fields:
   ```sql
   CREATE TABLE registration (
       id INT NOT NULL PRIMARY KEY AUTO_INCREMENT,
       username VARCHAR(50) NOT NULL UNIQUE,
       password VARCHAR(255) NOT NULL
   );
   ```
6. Access the project through your browser at: http://localhost/New_Lesson/
